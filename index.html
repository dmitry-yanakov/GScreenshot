Microsoft Authenticode

A digital signature for Windows that confirms the publisher. Its absence simply means the project is non-commercial and does not pay for annual Microsoft certificates (~$400/year).

Apple Notarization

Apple's security procedure where the Gatekeeper system automatically flags third-party apps as "unverified" even if they are safe. Obtaining a Gatekeeper certificate costs (~$99/year).

How to Launch the App

A quick guide to navigating system security restrictions

Windows (SmartScreen)

01. Click "More info" in the security window.
02. Click the "Run anyway" button that appears.
03. The app will launch and be added to your system's whitelist.

macOS (Gatekeeper)

01.
Copy the command below into the Terminal:
sudo xattr -rd com.apple.quarantine
02. Drag the GScreenshot icon into the Terminal window (this adds the file path).
03. Press Enter and provide your password if prompted.
